# Obsidian 使用教程

## 实用技巧

### 1. 快速创建笔记

- 输入 `[[新笔记名称]]` 并点击创建
- 使用命令面板（Ctrl/Cmd + P）

### 2. 快速格式化

- 选中文本使用快捷键：
	- Ctrl/Cmd + B：粗体
		- Ctrl/Cmd + I：斜体
		- Ctrl/Cmd + K：链接

### 3. 文件管理

- 使用文件夹组织笔记
- 创建模板文件（在.obsidian/templates 中设置）
- 使用星标收藏重要笔记

### 4. 搜索技巧

- 使用 `tag:#标签` 搜索标签
- 使用 `path:"文件夹"` 搜索路径
- 使用 `-搜索词` 排除内容

### 5. 插件推荐

1. **Dataview**：高级查询和数据分析
2. **Templater**：强大的模板功能
3. **QuickAdd**：快速添加内容
4. **Calendar**：日历视图
5. **Excalidraw**：手绘风格绘图
常用快捷键

### 通用快捷键

- `Ctrl/Cmd + N`：新建笔记
- `Ctrl/Cmd + O`：打开笔记
- `Ctrl/Cmd + S`：保存
- `Ctrl/Cmd + F`：在当前笔记中搜索
- `Ctrl/Cmd + Shift + F`：在所有笔记中搜索

### 编辑快捷键

- `Ctrl/Cmd + B`：粗体
- `Ctrl/Cmd + I`：斜体
- `Ctrl/Cmd + K`：链接
- `Ctrl/Cmd +]`：增加缩进
- `Ctrl/Cmd + [`：减少缩进

### 视图快捷键

- `Ctrl/Cmd + E`：切换编辑/预览模式
- `Ctrl/Cmd + Shift + E`：切换实时预览

## 最佳实践

### 1. 命名规范

- 使用有意义的文件名
- 避免特殊字符
- 使用连字符而不是空格

### 2. 组织结构

- 创建清晰的文件夹结构
- 使用标签进行跨文件夹分类
- 定期整理和归档

### 3. 链接策略

- 创建双向链接
- 使用 MOC（内容地图）组织相关笔记
- 避免过度链接

### 4. 备份策略

- 定期备份 vault
- 使用 Git 进行版本控制
- 同步到云存储
